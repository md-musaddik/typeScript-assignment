How do the four pillars of OOP—Inheritance, Polymorphism, Abstraction, and Encapsulation—help manage logic and reduce complexity in large-scale TypeScript projects?

The four pillar of OOP  help developers manage complex logic in large TypeScript projects by improving code organization. It also provides more reusability and maintainability.
Inheritance allow classes to reuse common properties and methods.
Polymorphism enable different classes to implement the same method in different ways.That makes applications more flexible.
